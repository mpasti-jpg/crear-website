# CREAR landing page

Statyczna strona internetowa przygotowana na podstawie projektu Figma:
`https://www.figma.com/design/pprFSvIQcsnNM13kRpOSqc/web-design-2026?node-id=2624-6038&m=dev`

## Uruchomienie

```bash
node server.js
```

Strona będzie dostępna pod adresem:

```text
http://127.0.0.1:4173
```

Jeśli w środowisku jest dostępne `npm`, można też użyć:

```bash
npm start
```

## Pliki

- `index.html` - struktura strony i treści.
- `styles.css` - layout, responsywność, animacje i odwzorowanie stylu Figma.
- `script.js` - animacje wejścia, menu mobilne i przycisk powrotu na górę.
- `server.js` - prosty lokalny serwer bez zależności.
